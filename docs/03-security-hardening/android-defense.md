# Android Security Hardening

## Google Play Protect

```
Settings > Security > Google Play Protect > Enable
- Scan apps with Play Protect
- Improve harmful app detection
```

## Permission Management

```
Settings > Apps > Permissions Manager
Review and minimize:
- Location
- Camera
- Microphone
- SMS
- Phone
```

## Unknown Sources

```
Settings > Security > Install unknown apps > Disable for all apps
```

## Safe Mode

Test if malware present:
```
Power off > Power on > Hold Volume Down during boot
```

If issues disappear in Safe Mode, third-party app is cause.

## GrapheneOS (High-Risk Users)

Privacy-focused Android variant:
- Enhanced security features
- Minimal Google services
- Regular security updates
- Installation requires unlocked bootloader

## Regular Updates

```
Settings > System > System update > Check for update
```

Enable automatic updates if available.
