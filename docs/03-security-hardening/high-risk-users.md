# High-Risk User Guidance

## Who is High-Risk?

- Journalists covering sensitive topics
- Human rights activists
- Political opposition members
- Lawyers representing controversial clients
- Whistleblowers
- Dissidents under authoritarian regimes

## Enhanced Security Measures

### Device Selection

**Consider:**
- Latest iOS device (better security, faster updates)
- OR GrapheneOS on supported Android
- Avoid older devices with unpatched vulnerabilities

### Mandatory Hardening

1. **Enable Lockdown Mode** (iOS) or GrapheneOS
2. **Disable iMessage** or use only with trusted contacts
3. **VPN always-on** with kill switch
4. **Regular device rotation** (6-12 months)
5. **Dedicated devices** for sensitive work

### Communication Strategy

**Sensitive Communications:**
- In-person when possible
- Signal on dedicated device
- Avoid phone calls for sensitive topics
- Never SMS for sensitive content

**Assume Surveillance:**
- Self-censor on regular devices
- Use code words/euphemisms (limited effectiveness)
- Meet in person for critical discussions

## Travel Security

**Before Travel:**
- Use burner device
- Transfer minimal data
- Document device state
- Share travel plans with trusted contacts

**During Travel:**
- Keep device with you always
- Power off when not in use
- Avoid connecting to unknown networks
- Be aware of border searches

**After Travel:**
- Forensic analysis on return
- Factory reset travel device
- Resume use of regular device

## Incident Response Plan

**If Infection Suspected:**

1. Stop using device immediately
2. Enable airplane mode
3. Contact:
   - Amnesty International Security Lab
   - Citizen Lab
   - Access Now Digital Security Helpline
4. Preserve evidence
5. Use backup communication channel

**Trusted Resources:**
- Amnesty Tech: security@amnesty.org
- Citizen Lab: info@citizenlab.ca
- Access Now: https://www.accessnow.org/help/

## Legal Considerations

- Know your rights regarding device searches
- Attorney-client privilege considerations
- Source protection for journalists
- Consult with legal experts

## Psychological Impact

Surveillance awareness can cause:
- Anxiety and stress
- Self-censorship
- Paranoia

**Support Resources:**
- Mental health professionals familiar with digital security
- Peer support groups
- Organizations supporting at-risk journalists/activists

## Key Recommendations

1. **Take Threat Seriously** - Nation-state surveillance is real
2. **Defense in Depth** - Multiple overlapping controls
3. **Compartmentalize** - Separate sensitive from general use
4. **Stay Informed** - Follow Citizen Lab, Amnesty Tech reports
5. **Seek Support** - You're not alone, resources available
