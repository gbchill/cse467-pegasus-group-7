### Key Terms
- **Zero-Click:** Exploit requiring no user interaction
- **IOC:** Indicator of Compromise
- **RCE:** Remote Code Execution
- **ASLR:** Address Space Layout Randomization
- **Sandbox:** Isolation mechanism for apps
